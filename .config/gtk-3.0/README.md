# GTK-3.0 and GTK-4.0 Theme Configuration

This directory contains GTK theme configurations that provide consistent theming across GTK applications, including Thunar file manager.

## Files

### settings.ini (GTK-3.0)
Main GTK theme configuration:
- **Theme**: Adwaita-dark (dark theme enabled)
- **Font**: Adwaita Sans 11
- **Cursor**: Adwaita theme, size 24
- **Rendering**: Antialiasing, slight hinting
- **Toolbar**: Horizontal with large icons

### dank-colors.css (GTK-3.0 and GTK-4.0)
Custom color scheme generated by Matugen:
- **Accent Color**: Purple (#e5b6f2) - matches niri/DMS theme
- **Background**: Dark (#161217)
- **Foreground**: Light (#eae0e7)
- **Consistent colors** across window, headerbar, popover, view, card, and sidebar

This color scheme ensures visual consistency between:
- Niri window manager
- DMS (DankMaterialShell)
- Thunar file manager
- All GTK applications

### bookmarks (GTK-3.0)
Default file manager bookmarks:
- Documents
- Music
- Pictures
- Videos
- Downloads

These appear in the sidebar of Thunar and other GTK file managers.

## Thunar Configuration

Thunar uses these GTK configurations for theming, plus its own specific settings:

### Thunar/accels.scm
Keyboard shortcuts for Thunar file manager operations.

### Thunar/uca.xml
Custom actions configuration for right-click context menu in Thunar.

### xfce4/xfconf/xfce-perchannel-xml/thunar.xml
Thunar application settings:
- View preferences (icon view, zoom level)
- Window size and position
- Show hidden files preference
- Sidebar width

## How It's Applied

During installation, the install.sh script copies these files to:
- `~/.config/gtk-3.0/` - GTK-3.0 configuration
- `~/.config/gtk-4.0/` - GTK-4.0 configuration
- `~/.config/Thunar/` - Thunar-specific configuration
- `~/.config/xfce4/xfconf/xfce-perchannel-xml/` - Xfce configuration

## Customization

**To change GTK theme:**
Edit `settings.ini` and change `gtk-theme-name` to your preferred theme.

**To change colors:**
Edit `dank-colors.css` in both gtk-3.0 and gtk-4.0 directories. The color scheme uses standard GTK color names:
- `accent_color` - Accent color for selected items
- `window_bg_color` - Window background
- `window_fg_color` - Window foreground (text)
- `headerbar_bg_color` - Titlebar background
- etc.

**To add bookmarks:**
Edit `bookmarks` file and add lines in format: `file:///path/to/directory`

## Notes

- GTK-3.0 and GTK-4.0 configurations are separate but should use matching color schemes
- The custom color scheme overrides the base Adwaita-dark theme colors
- Colors are defined using hex codes (#RRGGBB format)
- All GTK applications (Thunar, text editors, etc.) will use these theme settings
