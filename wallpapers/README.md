# Desktop Wallpapers

This directory contains 11 desktop wallpapers that will be installed to `~/Pictures/Wallpaper/` during setup.

## Wallpaper Browser

After installation, you can browse and change wallpapers using the DMS wallpaper browser:

- **Keybinding:** `Mod+Y`
- Or use: `dms ipc call dankdash wallpaper`

## Adding Custom Wallpapers

To add your own wallpapers:

1. Copy image files to `~/Pictures/Wallpaper/`
2. Open the DMS wallpaper browser (`Mod+Y`)
3. Select your desired wallpaper

Supported formats: JPG, PNG, and other common image formats

## Wallpaper Fill Mode

The default fill mode is set to "Fill" in DMS settings. You can change this in:
- DMS Settings → Appearance → Wallpaper Fill Mode
- Options: Fill, Fit, Stretch, Center, Tile
